# Beautiful World | 3D Scroll Experience

An immersive, cinematic web experience that utilizes CSS 3D transforms and vanilla JavaScript to create a deep parallax scrolling effect. 

## 🚀 Features

- **3D Parallax Navigation:** Content moves through the Z-axis as the user scrolls, creating a sense of physical depth.
- **Immersive Media:** Integrated background videos and high-resolution imagery optimized for the web.
- **Ambient Soundscape:** Includes a custom audio controller with auto-pause functionality when the browser tab loses focus.
- **Performance Optimized:** Uses `will-change` properties and optimized JavaScript loops to ensure smooth 60fps animations.
- **Responsive Design:** Utilizes viewport units (`vw`, `vh`) and CSS variables for a fluid layout across different screen sizes.

## 🛠️ Technical Stack

- **HTML5**: Semantic structure and media elements.
- **CSS3**: 
    - `perspective` and `preserve-3d` for the spatial container.
    - `translateZ` for the depth positioning.
    - Custom properties (CSS variables) for design consistency.
- **Vanilla JavaScript**: 
    - Dynamic scroll calculation and style injection.
    - Audio API management and window state event listeners.

- `/css`: Contains `main.css` for 3D layout and styling.
- `/js`: Contains `app.js` for scroll logic and audio handling.
- `/media`: Optimized video and audio assets.
- `/fonts`: Custom typography (Raleway).
